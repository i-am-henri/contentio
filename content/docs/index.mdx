# Getting Started

Contentio is a modern CLI to add content to an NextJS Application. This works with <Link href="https://mdxjs.com">MDX</Link> and <Link href="https://github.com/hashicorp/next-mdx-remote">next-mdx-remote</Link>.

<br />
<br />
## Get started with this command:
<br />
<Tabs
  customId="install"
  tabs={[
    {
      name: "npm",
      content: <Viewer code="npx contentio@latest init" language="bash" />,
    },
    {
      name: "bun",
      content: (
        <Viewer code="bunx --bun contentio@latest init" language="bash" />
      ),
    },
  ]}
/>
<br />
<br />
