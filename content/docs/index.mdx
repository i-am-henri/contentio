# Contentio Docs
Contentio is a modern CLI to add content to an NextJS Application. This works with <Link href="https://mdxjs.com">MDX</Link> and <Link href="https://github.com/hashicorp/next-mdx-remote">next-mdx-remote</Link>.
<br/>
### Get started with this command:
<br />
npm:
<Viewer code={`npx contentio@latest init`} language="bash" />
<br />
If you want to get an overview ran this command:
<Viewer code={`npx contentio@latest`} language="bash" />
Or:
<Viewer code={`npx contentio@latest help`} language="bash" />
<br />
If you ran this command, you should get this file structure:
